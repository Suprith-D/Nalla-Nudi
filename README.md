Nalla-Nudi

Nalla-Nudi is an Android application developed to help Kannada-medium students understand technical English vocabulary used in subjects such as Science, Mathematics, and Commerce. The application acts as a bridge dictionary by providing simple Kannada explanations, pronunciation guidance, and subject-based filtering for technical terms.

Features
Technical English to Kannada glossary
Subject-wise filtering
Search functionality
Text-to-Speech pronunciation support
Offline access using Room Database
User-friendly interface
“My List” feature to save difficult words
Problem Statement

Many Kannada-medium students face difficulty understanding technical English vocabulary used in academics. This language barrier affects learning confidence and subject understanding. Existing dictionary applications do not focus on technical educational terms or subject-based explanations.

Nalla-Nudi solves this problem by providing easy Kannada explanations, pronunciation support, and offline learning access through a simple Android application.
